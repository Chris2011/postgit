Configuration

In order to create a project in a Gitlab, first of all is to complete configuration page:

  • URL: The url of the Gitlab server. (ex: http://192.168.254.247)
  • User: Your user in Gitlab server.
  • Password: Your password in Gitlab server.
  • URL Hook: If you want to create a Tag push event hook in your project, insert here the complete destination url.
  • Developers: This list shows the users created in Gitlab. Selected users will be added as developers. (To see the users on the list; first, you must enter URL, User, Password and click Apply or OK)
